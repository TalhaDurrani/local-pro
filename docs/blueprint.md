# **App Name**: ProLocal

## Core Features:

- Secure Dual-Role Authentication: Unique phone-number based signup using Supabase Auth with dynamic role switching between service seekers and providers.
- Location-Aware Onboarding: Automatic address population using browser geolocation and reverse-geocoding coordinates for precise service mapping.
- AI Professional Bio Tool: A generative AI tool used by providers to refine their service biographies based on experience and category.
- Integrated Wallet Billing: Simulation of JazzCash and Easypaisa payment processing that updates subscription status and activates provider visibility.
- Service Request Pipeline: A status-driven system for users to hire, track, and complete service requests with providers using a centralized database architecture.
- Provider Performance Metrics: Dashboard for providers to view their services delivered, average rating, and real-time trial expiration status.
- Superadmin Global Kill Switch: High-security administrative panel with a responsive override toggle for instant global trial termination across the platform.
- Bilingual Interface Support: Fully localized user interface supporting both English and Urdu languages to ensure accessibility for all users.

## Style Guidelines:

- Primary color: Dark Slate Grey (#354f52) for a professional and sophisticated aesthetic.
- Accent color: Sage Green (#cad2c5) used for highlights and soft interactive elements.
- Deep Contrast color: Charcoal Grey (#2f3e46) for typography and deep structural elements.
- Glassmorphism design language featuring high backdrop-blur (16px), semi-transparent surfaces (opacity 15%), and subtle white borders.
- Strict mobile-first responsive layout to accommodate service providers using handheld devices in the field.
- Bilingual typography pairing 'Space Grotesk' for modern English headings and 'Noto Nastaliq Urdu' for elegant, readable Urdu text.
- Minimalist, sharp line icons to reinforce a high-tech utility feel across all service categories.
- Smooth micro-interactions for billing states and language toggles, utilizing subtle fade and blur transitions.
